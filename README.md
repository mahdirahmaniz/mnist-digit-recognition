# mnist-digit-recognition
# تشخیص اعداد دست‌نویس با TensorFlow

## پروژه چیست؟
یک شبکه عصبی ساده با TensorFlow/Keras که اعداد دست‌نویس (۰ تا ۹) را تشخیص می‌دهد.

## دقت مدل
بین ۹۰ تا ۹۹ درصد روی دیتاست MNIST

## تکنولوژی‌ها
- Python
- TensorFlow / Keras
- NumPy

## نحوه اجرا
1. فایل Tensor.py را در VScode باز کنید
2. مسیر دیتا ها را اصلاح کنید
3. کد را در اجرا کنید 

## نتیجه
مدل با دقت بالا اعداد را تشخیص می‌دهد.
